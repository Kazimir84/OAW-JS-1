    - Визначити відповідний оператор в виразах що б вийшов відповідний результат. В однакових виразах не використовувати однакові оператори!!!

  • (5 < 6) -> true
  • (5 == 6) -> false
  • (5 > 6) -> false

  • (10 === 10) -> true

  • (10 > 10) -> false
